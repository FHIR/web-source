The FHIR R4 Cross-Version Support Package (hl7.fhir.xver.r4)

A bunch of useful resources for converting between R4 and other FHIR versions.

Releases

  • 1.2.0: Add Version specific Structure Definitions (2020-03-13)
  • 1.1.0: Add Parameter name and search Parameter Mappings (2020-03-02)
  • 1.0.0: First release (based on R3/R4 published standards) (2020-03-01)

Maintaining the Package

The content of the package is maintained in the FHIR Packages repository at https://github.com/FHIR/packages/tree/master/packages/hl7.fhir.xver.r4
