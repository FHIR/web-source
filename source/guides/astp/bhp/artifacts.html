This page provides a list of the FHIR artifacts defined as part of this implementation guide.

Profiles

Mental Health Clinical Notes Profile

This profile on US Core DocumentReference constrains type to be of five allowable mental health clinical note types via binding to Mental Health Clinical Notes (ValueSet)
Grant Information Observation Profile

This profile defines the structure and terminology for Observations related to Program ID, number, and funding type. Feedback is requested throughout this profile, specifically do standard concepts exist for .category, .code, and .value[x]? Also, use of a complex extension to represent both grant number and program ID.
Grant Program ID and Number Extension

This complex extension defines the representation of Grant Program ID as well as grant number.

Terminology

Mental Health Clinical Notes Profile

This ValueSet defines the LOINC codes for Mental Health Clinical Note types.
Behavioral Health Program Funding Types

This ValueSet defines coded concepts pertaining to a Behavioral Health program/grant.
Behavioral Health Program Funding Type Codes

This CodeSystem defines coded concepts pertaining to a Behavioral Health program/grant.
