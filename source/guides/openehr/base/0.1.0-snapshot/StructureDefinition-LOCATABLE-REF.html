Logical Model: LOCATABLE_REF

Official URL: http://openehr.org/fhir/StructureDefinition/LOCATABLE-REF Version: 0.1.0-snapshot
Standards status: Informative Maturity Level: 3 Computable Name: LOCATABLE_REF

Reference to a LOCATABLE instance inside the top-level content structure inside a VERSION identified by the id attribute. The path attribute is applied to the object that VERSION.data points to

Original Source: specifications.openehr.org

Usages:

  • Use this Logical Model: INSTRUCTION_DETAILS and PARTY
