Logical Model: DV_QUANTIFIED ( Abstract )

Official URL: http://openehr.org/fhir/StructureDefinition/DV-QUANTIFIED Version: 0.1.0-snapshot
Standards status: Informative Maturity Level: 3 Computable Name: DV_QUANTIFIED

Abstract class defining the concept of true quantified values, i.e. values which are not only ordered, but which have a precise magnitude.

Original Source: specifications.openehr.org

Usages:

  • Derived from this Logical Model: DV_ABSOLUTE_QUANTITY and DV_AMOUNT

DV_QUANTIFIED C 0..* DV_ORDERED This is an abstract type. Child types: DV_ABSOLUTE_QUANTITY, DV_AMOUNT
Abstract class defining the concept of true quantified values, i.

Elements defined in Ancestors:CR, LF, Local_terminology_id, normal_status, normal_range, other_reference_ranges
Constraints: Other_reference_ranges_validity, Is_simple_validity, Normal_range_and_status_consistency, Magnitude_status_valid
CR 1..1 string Carriage return character.
LF 1..1 string Line feed character.
Local_terminology_id 1..1 string Predefined terminology identifier
normal_status 0..1 CODE_PHRASE Optional normal status indicator of value with respect to normal range for this value.
normal_range 0..1 DV_INTERVAL Optional normal range.
other_reference_ranges 0..* REFERENCE_RANGE<DV_ORDERED> Optional tagged other reference ranges for this value in its particular measurement context.
magnitude_status 0..1 string Optional status of magnitude with values: * `"="` : magnitude is a point value * `"<"` : value is < magnitude * `">"` : value is > magnitude * `"<="` : value is <= magnitude * `">="` : value is >= magnitude * `"~"` : value is approximately magnitude If not present, assumed meaning is `"="` .
accuracy 0..1 Base Accuracy of measurement.

Documentation for this format

Constraints

Id Grade Path(s) Description Expression
Is_simple_validity error DV_QUANTIFIED Is simple validity must satisfy conditional constraints (normal_range .empty() and other_reference_ranges .empty()) implies is_simple()
Magnitude_status_valid error DV_QUANTIFIED Magnitude status valid must be present magnitude_status.exists() implies valid_magnitude_status() (magnitude_status)
Normal_range_and_status_consistency error DV_QUANTIFIED Normal range and status consistency must be present (normal_range.exists() and normal_status.exists()) implies (normal_status.code_string.is_equal (“N”) xor normal_range.not().has (self))
Other_reference_ranges_validity error DV_QUANTIFIED Other reference ranges validity must be present other_reference_ranges.exists() implies other_reference_ranges.empty().not()
