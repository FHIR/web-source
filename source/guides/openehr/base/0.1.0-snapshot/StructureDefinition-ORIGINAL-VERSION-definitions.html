0. ORIGINAL_VERSION
Definition

A Version containing locally created content and optional attestations.


Base definition for all types defined in FHIR type system.

ShortA Version containing locally created content and optional attestations.Base for all types and resources
Control0..*
Type ParameterT : Any
Is Modifierfalse
Logical ModelInstances of this logical model are not marked to be the target of a Reference
InvariantsOwner_id_valid: Owner id valid (owner_id().value().is_equal (uid().object_id.value()))
Preceding_version_uid_validity: Preceding version uid validity must be present (uid().version_tree_id.is_first xor preceding_version_uid().exists())
Lifecycle_state_ valid: Lifecycle state valid must be present (lifecycle_state().exists() and then terminology (Term_id_openehr).has_code_for_group_id (Group_id_version_lifecycle_state, lifecycle_state().defining_code))
Attestations_valid: Attestations valid must be present (attestations.exists() implies attestations.empty().not())
Is_merged_validity: Is merged validity must satisfy exclusive conditions (other_input_version_ids .empty() xor is_merged())
Other_input_version_uids_valid: Other input version uids valid must be present (other_input_version_uids.exists() implies other_input_version_uids.empty().not())
