0. RESOURCE_DESCRIPTION_ITEM
Definition

Language-specific detail of resource description. When a resource is translated for use in another language environment, each RESOURCE_DESCRIPTION_ITEM needs to be copied and translated into the new language.


Base definition for all types defined in FHIR type system.

ShortLanguage-specific detail of resource description.Base for all types and resources
Control0..*
Is Modifierfalse
Logical ModelInstances of this logical model are not marked to be the target of a Reference
InvariantsPurpose_valid: Purpose valid must not be empty (purpose.empty().not())
Use_valid: Use valid must be present (use.exists() implies use.empty().not())
misuse_valid: misuse valid must be present (misuse.exists() implies misuse.empty().not())
copyright_valid: copyright valid must be present (copyright.exists() implies copyright.empty().not())
