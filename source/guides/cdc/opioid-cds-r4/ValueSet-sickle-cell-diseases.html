Sickle-cell disorders that cause painful crisis

Conditions due to Sickle-cell that can cause painful crisis to be used as exclusions for opioid use recommendations

References

This value set is not used here; it may be used elsewhere (e.g. specifications and/or implementations that use this content)

Logical Definition (CLD)

Profiles: CPGComputableValueSet, CPGExecutableValueSet

This value set includes codes based on the following rules:

  • Include codes fromhttp://snomed.info/sct version 📍http://snomed.info/sct/731000124108 where concept is-a 417357006 (Sickling disorder due to hemoglobin S (disorder))
  • Include codes fromhttp://snomed.info/sct version 📍http://snomed.info/sct/731000124108 where concept is-a 441482006 (History of sickle cell anaemia)
  • Include codes fromhttp://hl7.org/fhir/sid/icd-10-cm version Not Stated (use latest from terminology server) where concept is-a D57 (Sickle-cell disorders)

This value set excludes codes based on the following rules:

  • Exclude codes fromhttp://snomed.info/sct version 📍http://snomed.info/sct/731000124108 where concept is-a 416290001 (Hemoglobin S sickling disorder without crisis (disorder))
  • Exclude codes fromhttp://hl7.org/fhir/sid/icd-10-cm version Not Stated (use latest from terminology server) where concept is-a D57.1 (Sickle-cell disease without crisis)
