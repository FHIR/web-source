To register an implementation guide, in the Implementation Guide Registry, please do one of the following:

  • Make a pull request directly to the FHIR IG Registry GitHub repository (preferred)
  • Fill the form in below (submissions are emailed to fhir-director@hl7.org for review)
  • Or just email fhir-director@hl7.org directly (e.g. if your implementation guide is multi-lingual or uses a language not listed below)
Basic Information
The full name of the implementation guide
The NPM package identifier
The organization responsible for this IG
Two-letter country code (use 'uv' for universal/international)
URLs
The canonical base URL for this implementation guide
URL to the version history page (optional)
URL to the continuous integration build (optional)
Edition Information
The name/label for this edition
The implementation guide version number
Full package identifier (typically npm-name#version)
Direct URL to this specific edition
