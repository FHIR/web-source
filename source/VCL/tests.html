üìñ About VCL Parser

This page tests the JavaScript implementation of the Value Set Composition Language (VCL) Parser.

VCL Syntax Overview:

  • , - Conjunction (AND)
  • ; - Disjunction (OR)
  • - - Exclusion (NOT)
  • () - Grouping
  • << - IS-A relationship
  • ~<< - IS-NOT-A relationship
  • = - Equals
  • ^ - Include ValueSet
  • * - Wildcard (all concepts)
  • "text" - Quoted strings
